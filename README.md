# 설교문 말씀 추출기

설교문 DOCX 파일에서 성경 본문과 보조 본문 구절을 자동으로 감지하여 색상 표시하고, 참조 구절을 정리해 주는 웹 툴입니다.

> 설교문의 내용을 저장하거나 외부로 전송하지 않습니다.  
> 성경 말씀 구절 참조(예: `28:1`, `마 27:51`)만 추출합니다.

---

## 사용 방법

**접속 URL:** `http://192.168.35.80:8000/`

1. DOCX 설교문 파일을 드래그하거나 클릭하여 업로드
2. **처리 시작** 클릭
3. ZIP 다운로드 → DOCX / PDF / MD 파일 포함

---

## 처리 규칙

| 단락 패턴 | 처리 방식 | 분류 |
|---|---|---|
| `28:1`, `28:5-7` 등 숫자:숫자로 시작 | 🔴 빨간색 | 본문 |
| `마 27:51`, `고전 15:4` 등 책이름 숫자:숫자로 시작 | 🟡 노란 하이라이트 + 밑줄 | 보조본문 |
